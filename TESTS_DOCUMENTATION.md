# 📋 Документация по юнит тестам BeerTestShop

## Обзор

Проект содержит комплексные юнит тесты для всех REST контроллеров и сервисов приложения интернет-магазина пива.

## Структура тестов

```
src/test/java/com/beertestshop/
├── controller/
│   ├── ProductControllerTest.java    # 15 тестов
│   ├── FeedbackControllerTest.java   # 14 тестов
│   └── CartControllerTest.java       # 9 тестов
└── service/
    ├── ProductServiceTest.java       # в работе
    ├── FeedbackServiceTest.java      # в работе
    └── CartServiceTest.java          # в работе
```

## Запуск тестов

### Запуск всех тестов
```bash
mvn test
```

Результат:
```
[INFO] Tests run: $, Failures: 0, Errors: 0, Skipped: 0
[INFO] BUILD SUCCESS
```

### Запуск тестов конкретного контроллера
```bash
mvn test -Dtest=ProductControllerTest
mvn test -Dtest=FeedbackControllerTest
mvn test -Dtest=CartControllerTest
```

### Запуск с покрытием кода (JaCoCo)
```bash
mvn clean test jacoco:report
```

Отчет будет доступен в: `target/site/jacoco/index.html`

## Используемые технологии тестирования

- **JUnit 5** - фреймворк для юнит тестирования
- **Mockito** - мокирование зависимостей
- **Spring Boot Test** - тестирование Spring компонентов
- **MockMvc** - тестирование REST контроллеров
- **AssertJ** - fluent assertions для читаемых проверок
- **Jackson ObjectMapper** - сериализация JSON для тестов
